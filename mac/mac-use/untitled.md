# ibook文件存放路径

进入到ibook实际存放文件地址：

```text
cd ~/Library/Containers/com.apple.BKAgentService/Data/Documents/iBooks/Books
```

将原ibook地址软连接到~/ibook  ：

```text
ln -s ~/Library/Containers/com.apple.BKAgentService/Data/Documents/iBooks/Books ~/ibook
```
